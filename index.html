🛠 Feature Overview

Skylandia packs an extensive set of tools for explorers, PvP tacticians, builders, and automation enthusiasts. Modules are built to be flexible, powerful, and user‑friendly.

🎮 How Modules Are Used

Each module in Skylandia is toggled via the Meteor Client GUI. You can configure settings, bind keys, and interact with modules in-game. Below, each module is described with usage examples.

🧭 Exploration & Hunting

  • ActivatedSpawnerDetector
    Pinpoints active mob spawners for lucrative hunts.
    "I want to find mob spawners for XP farms."
    "Enable ActivatedSpawnerDetector and fly around caves. Spawners will be highlighted when active."
  • CaveDisturbanceDetector
    Detects signs of recent player activity to locate hidden stashes or bases.
    "Are there any bases nearby?"
    "Turn on CaveDisturbanceDetector to scan for player-placed blocks and disturbances."
  • Trails
    Records and replays your movement paths for analysis or nostalgia.
    "I want to retrace my steps from last week."
    "Enable Trails to record your path, then replay it later."
  • BaseFinder
    Identifies base-like structures using block/entity patterns.
    "Is this chunk a possible base?"
    "BaseFinder will scan for signs like signs, portals, and bubble columns."
  • StashBotModule
    Auto-logs and manages stashes with waypoints and alerts.
    "I found a stash, can I mark it?"
    "StashBotModule will log the stash and set a waypoint automatically."
  • BetterStashFinder
    Adds refined filters and Xaero's Minimap/World Map integration.
    "I want to filter stashes by loot type."
    "BetterStashFinder lets you set custom filters and integrates with Xaero's maps."
  • OldChunkNotifier
    Alerts when entering historically loaded chunks.
    "Am I in an old chunk?"
    "OldChunkNotifier will notify you when you enter a chunk with history."
  • TrailFollower
    Automatically retraces discovered trails via pitch-flight or Baritone.
    "Can I follow someone's trail?"
    "TrailFollower will automate your movement along recorded trails."
  • SearchBot
    Executes grid, spiral, random, or custom pattern searches with waypoint automation.
    "I want to search a large area for stashes."
    "SearchBot will move you in a search pattern and mark found locations."
  • TerrainAnalyzer
    Profiles terrain for strategic builds, travel, or PvP choke-points.
    "Is this terrain good for a base?"
    "TerrainAnalyzer will analyze and report strategic features."

🎮 Module Usage

For each module, open the Meteor Client GUI, find the module under its category, and toggle or configure as needed. Most modules have settings for customization and can be bound to hotkeys for quick access.
